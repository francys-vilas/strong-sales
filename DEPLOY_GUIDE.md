# Guia de Publicação Rápida

O build de produção foi gerado com sucesso!

## Opção Mais Rápida (Sem Login/Configuração)

1. Acesse **[Netlify Drop](https://app.netlify.com/drop)**.
2. Localize a pasta `dist` no seu projeto:
   `c:\repositorioflutter\Strong-sales\dist`
3. **Arraste a pasta `dist`** para a área pontilhada no site do Netlify.
4. O site ficará online imediatamente.

## Opção Via Linha de Comando (Vercel)

Se você tiver uma conta Vercel:

1. Abra o terminal na pasta do projeto.
2. Execute `npx vercel`.
3. Siga as instruções na tela.
